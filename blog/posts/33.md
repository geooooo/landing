Приведу примеры современных cli утилит, которые являются альтернативами стандартных GNU / Unix way утилит. Многие из них, к слову, написаны на Rust 🦀.

Само собой у них полно дополнительных опций. Все их можно найти на Github. 

Далее, современная альтернатива - исходник:
- bat - cat
- duf - df
- dust - du
- htop - top
- gping - ping
- procs - ps
- neovim - vim
- zellij - tmux
- yazi - файловый менеджер прямо в консоли
